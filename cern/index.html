cern.timber.get_from_timber(data, filename, fillnumber=0, arguments='', overwrite=False, dataformat='TSV', minfilesize=1000)[source]¶

Get a set of data from TIMBER and store it in file filename.dataformat

Parameters:
  • data (list) – Can also be comma-separated string. List of variable names to extract.
  • filename (string) – Name of file where data is stored, as filename.dataformat
  • fillnumber (integer) – Which fill number to extract from, otherwise latest data.
  • arguments (string) – Additional arguments to pass to the java command (expert option)
  • overwrite (bool) – if True, overwrites file if it already exist.
  • dataformat (string) – Type of formatting in the output file.
  • minfilesize (int) – Minimum size of file in bytes for successful extraction (expert option)
